## Response Format

All API responses follow a consistent format:

### Success Responses

```json
{
  "data": [...],  // Array of results or single object
  "pagination": {  // Only included for list endpoints
    "page": 1,
    "limit": 20,
    "total": 100,
    "totalPages": 5
  }
}
```

### Error Responses

```json
{
  "error": "Error message",
  "details": "Additional error details"
}
```

## Rate Limits

<Note>
  Our servers can handle a lot of requests, but even ducks need to come up for air sometimes! Please respect our rate limits.
</Note>

The DuckTape WMS API has the following rate limits:

- 100 requests per minute per API key
- 5,000 requests per day per API key

If you exceed these limits, you'll receive a `429 Too Many Requests` response.
